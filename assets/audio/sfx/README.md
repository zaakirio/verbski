# Sound Effects

This directory contains sound effects for the app.

## Current Files

✅ `success.mp3` - Played when user answers correctly (bling3.mp3)
✅ `error.mp3` - Played when user answers incorrectly (error1.mp3)

## Recommended Sources

### Free Sound Effect Libraries

1. **Freesound.org** (https://freesound.org/)
   - Search for "success chime" or "positive notification"
   - Search for "error beep" or "wrong answer"
   - License: Creative Commons (check individual licenses)

2. **Zapsplat** (https://www.zapsplat.com/)
   - Great collection of UI sounds
   - Free for personal and commercial use with attribution

3. **Mixkit** (https://mixkit.co/free-sound-effects/)
   - No attribution required
   - Search for "notification" and "error" sounds

### Characteristics for Duolingo-style Sounds

**Success Sound:**
- Duration: 0.3-0.6 seconds
- Style: Bright, uplifting chime or bell
- Tone: Major key, ascending notes
- Example searches: "success chime", "level up", "achievement"

**Error Sound:**
- Duration: 0.2-0.4 seconds
- Style: Gentle, not harsh or punishing
- Tone: Descending notes, softer
- Example searches: "wrong answer", "error beep", "negative notification"

## Installation

1. Download the sound effect files from one of the sources above
2. Convert to MP3 format if needed (use online converters like CloudConvert)
3. Rename files to `success.mp3` and `error.mp3`
4. Place them in this directory
5. Restart the app to load the new sound effects

## Notes

- Keep files small (< 50KB each) for fast loading
- Ensure volume levels are consistent with verb pronunciation audio
- Test with device muted to verify haptic feedback still works
